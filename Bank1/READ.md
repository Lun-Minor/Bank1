## Вывод
Проведено модульное тестирование класса `BankAccount`.  
Написаны и успешно выполнены автоматические тесты для Debit:
  - корректное списание суммы → баланс уменьшается (после изменений в коде на   m_balance -= amount;)
  - попытка списать отрицательную сумму → исключение `ArgumentOutOfRangeException`
  - попытка списать сумму больше баланса → исключение `ArgumentOutOfRangeException`
Написаны и успешно выполнены автоматические тесты для Credit:
  - корректное пополнение → баланс увеличивается
  - попытка внести отрицательную сумму → исключение `ArgumentOutOfRangeException`
Все тесты пройдены успешно.
